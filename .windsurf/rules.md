# benmilne.com — Windsurf agent rules

All benmilne.com API endpoints are public and unauthenticated.

## API endpoints

- REST API: `https://benmilne.com/api/posts`
- Search: `https://benmilne.com/api/search?q={query}`
- GraphQL: `POST https://benmilne.com/graphql`
- MCP server: `POST https://benmilne.com/mcp`
- OpenAPI spec: `https://benmilne.com/openapi.json`
